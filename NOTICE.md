# NOTICE

E-Review Agent 是一个毕业设计与学术研究演示项目，定位为“前后台一体化电商图文评论智能治理系统”。

本项目使用开源电商系统作为业务底座，在原有商品、订单、用户、评论和后台管理能力基础上进行二次开发。新增内容主要包括：

- 图文评论 AI 分析服务；
- AI Agent 自动巡检；
- 风险评论中心；
- 运营处理闭环；
- Agent Run / Step / Tool 可观测记录；
- Feedback 人工反馈闭环；
- Agent Eval 评估中心；
- FastAPI 专业 Agent 框架层；
- 模态冲突检测、主模态判断和 fallback 工程稳定性。

本项目不声称完整电商系统从零开发。对外发布、归档、答辩或论文引用时，应保留原始开源许可证、来源说明和本 NOTICE 文件。
