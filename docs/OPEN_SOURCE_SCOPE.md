# Open Source Scope

## Public

- Business source code for the litemall-based review governance system.
- FastAPI AI service source code.
- Rules, contracts, schemas and lightweight test fixtures.
- Deployment examples and Docker Compose configuration.
- Curated documentation and audit summaries.

## Not Public

- Model weights.
- Adapters, checkpoints and training outputs.
- Private datasets and raw training data.
- Full benchmark corpora.
- Real user data and database backups.
- Large indexes, including FAISS binaries.
- Private evidence and full soak raw logs.
- Screenshots containing personal information or local paths.

The public repository is a sanitized snapshot and does not contain the internal Git history.
